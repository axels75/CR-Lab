🔧 Migration vers Cloudflare D1

Configuration de la base de données pour CR Master en production

Étape 1 — Créer la base de données Cloudflare D1

ℹ️ Cloudflare D1 est une base de données SQLite gratuite, incluse dans votre plan Cloudflare Pages. Pas de coût supplémentaire.
1

Ouvrir le Dashboard Cloudflare

Aller sur dash.cloudflare.com → votre compte → Workers & Pages

2

Créer une base D1

Dans le menu gauche → D1 SQL Database → Create database

Nom de la base : cr-master-db

3

Lier D1 à votre projet Pages

Workers & Pages → votre projet cr-master → Settings → Functions

Section D1 database bindings → Add binding

Variable name : DB  |  D1 database : cr-master-db

Save puis redéployer le projet.

Étape 3 — Migrer vos données depuis Genspark

⚠️ Cette migration doit être effectuée depuis le sandbox Genspark (genspark.ai) où vous avez accès aux données. Elle copie toutes vos données vers D1.

Si vous êtes sur Cloudflare, cette migration ne peut pas accéder aux données Genspark (CORS). Ouvrez d'abord l'application dans le sandbox Genspark, connectez-vous, puis revenez ici.
